## Please follow these quick steps before pushing any code

## Setup

1. **Clone the repo**

    ```bash
   git clone <repo-url>
   cd <repo-name>
   ````

2. **Enable Git LFS**

   ```bash
   git lfs install
   git lfs pull
   ```

   > Train and test files are tracked using Git LFS â€” do not upload large files directly.

---

## Workflow

1. Create a new branch for your work:

   ```bash
   git checkout -b your-name/feature-name
   ```

2. Commit and push changes:

   ```bash
   git add .
   git commit -m "Update: short description"
   git push origin your-name/feature-name
   ```

3. Open a Pull Request to **main** for review.

---

Keep commits clean, avoid large file uploads, and sync regularly with `main`.
